# 🎵 LyricsAI

LyricsAI is an AI-powered web application built using Python, Flask, SQLite, and Google Gemini.

## Features

- AI Lyrics Analysis
- Song Information
- Mood Detection
- Genre Detection
- Analysis History
- Search History
- Delete History

## Tech Stack

- Python
- Flask
- SQLite
- Google Gemini API
- HTML
- CSS
- JavaScript
- Bootstrap
